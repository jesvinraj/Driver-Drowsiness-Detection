# Driver-Drowsiness-Detection
A Flask-based web application for detecting driver drowsiness using OpenCV and Haar Cascade classifiers.
A real-time web application that detects driver drowsiness using computer vision. Built with Flask and OpenCV, the system monitors eye movements through a webcam and triggers an alert when signs of drowsiness are detected, helping improve road safety.

## Features
- Real-time face and eye detection
- Drowsiness alert system
- User authentication (Login & Registration)
- Responsive web interface
- MySQL database integration

## Tech Stack
- Python
- Flask
- OpenCV
- HTML, CSS, JavaScript
- MySQL
